### To Run the Game:

```bash
npm run dev
```

The game will automatically open in your browser at http://localhost:3000

### Controls:

- **WASD** - Move around
- **Mouse** - Look/Aim
- **Left Click** - Shoot (click canvas first to lock cursor)
- **R** - Reload weapon
- **Shift** - Sprint
- **Space** - Jump

### What You Can Do:

1. **Movement**: Walk around the test arena using WASD
2. **Combat**: Shoot the red enemy soldiers (they patrol around the level)
3. **Exploration**: Navigate around boxes and walls (these provide cover)
4. **Survive**: Watch your health bar and ammo counter
